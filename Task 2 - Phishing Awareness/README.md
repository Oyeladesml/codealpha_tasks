# # CodeAlpha Cybersecurity Internship Portfolio

## Task 2: Phishing Awareness Training Platform

### 🌐 Project Overview
An interactive, enterprise-grade dark-themed web interface designed to educate users on detecting, deflecting, and reporting modern social engineering vectors. The platform transforms traditional static training content into an engaging, responsive experience featuring a custom evaluation engine.

### 🚀 Live Demonstration
* **Live Website Link:** https://oyeladesml.github.io/codealpha_tasks/

### 🛠️ Key Features Implemented
* **Comprehensive Vector Analysis:** Dedicated modules covering modern attack surfaces including Smishing (SMS), Vishing (Voice), and Quishing (QR Codes).
* **The SLAM Inspection Framework:** Actionable breakdown instructing users on how to systematically analyze Senders, Links, Attachments, and Message contexts.
* **Interactive "Spot the Phish" Quiz:** A built-in JavaScript-powered simulator presenting real-world email scenarios. It provides instant analytical feedback explaining safety flags or threat indicators based on header and protocol variations.
* **Responsive Architecture:** Fully optimized with custom CSS media queries to ensure layout compatibility across mobile, tablet, and desktop viewports.

### 📁 Technologies Used
* HTML5
* CSS3 (Flexbox, Grid System, Custom Animations)
* JavaScript (Vanilla ES6 State Engine)
* FontAwesome Icons & Google Fonts Integration
